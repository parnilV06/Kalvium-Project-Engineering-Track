# LeetCode Question Tracker

## What It Is
A web app to log and track LeetCode questions I solve, including code, status, and problem tags for easy revision.

## The Problem It Solves
I regularly solve LeetCode problems but often forget which questions I’ve already done, where I saved my solutions, and what approach I used. This app lets me store each question along with its code and tags so I can quickly revisit and revise them without searching through past submissions.

## What I Intentionally Excluded
- User authentication: Adding login would require session handling or JWT-based authentication, which would significantly increase backend complexity for an MVP that is intended for personal use only.

## Tech Stack
- Backend: Node.js + Express + SQLite (better-sqlite3)
- Frontend: Vanilla HTML, CSS, JavaScript
- Deployed: Render (backend) + Netlify (frontend)
